params.py - This does most of the work for interactive mode
	Python program to generate parameter file omg.ffr.  First, it reads previous
	run parameters from final (commented) line of existing omg.ffr.  Then,
	allows the user to modify run parameters to their heart's content.  Finally,
	copies basic parameter file ../../include/omg_0.ffr to ./omg.ffr and adds
	interesting parameters.  If, for some reason you lose the existing parameter
	file omg.ffr, there's a backup in MP_comgeant_minimal/include/.

To run ...
- Initiate comgeant parameters
	in MP_comgeant_minimal/run/interactive/ enter "./params.py" (without quotes)
	follow the instructions to generate parameter file omg.ffr
- Start interactive comgeant
	in MP_comgeant_minimal/run/interactive/ enter "./comg_inter.exe"
- for workstation enter "3"
- Initiate Moller Polarimeter layout
	at GEANT> prompt enter "exe plot_setup proj=(1,2,3)"
		proj=1: side view
		proj=2: plan view
		proj=3: 3D view
- Generate particle trajectories
	at GEANT> prompt enter "exe eve n=10"
	or adjust n to the desired number of events
- Exit GEANT
	at GEANT> prompt enter "exit"
